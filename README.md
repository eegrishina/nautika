## Описание

Многостраничный сайт для яхт-школы НАУТИКА в Новосибирске.  
Проект разработан с использованием **React**, функциональных компонентов и расширений **JSX**. Применяются React-хуки, JavaScript-функции и обработка событий.  
В работе также использовались **React Router** для маршрутизации и **HashRouter** для обеспечения корректной работы путей на GitHub Pages.  

Сайт адаптирован под различные устройства с помощью **CSS** и **медиа-запросов**. 

> Проект продолжает находиться в стадии разработки.

## Запуск приложения

1. Клонировать репозиторий:  
   ```  
   git clone <URL>  
   ```  
2. Установить зависимости:  
Убедитесь, что у вас установлен **Node.js** и **npm**. После этого выполните команду для установки всех зависимостей:
   ```  
   npm install  
   ```  
3. Запустить проект:  
   ```  
   npm start  
   ```  
   Приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000) в вашем браузере.
